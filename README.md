# OWL AI Research Internship Tasks

This repository contains the tasks completed as part of the OWL AI Research Internship.

## Task 1: AI/ML Researcher Role
Research document explaining the responsibilities, tools, and importance of an AI/ML researcher.

## Task 2: Large Language Models (LLMs)
Research on models such as ChatGPT, Gemini, Claude, and DeepSeek, including their working, applications, and limitations.

## Task 3: Exploratory Data Analysis (EDA)
Performed data analysis on a public dataset using Python, Pandas, Matplotlib, and Seaborn.

## Task 4: Machine Learning Model Comparison
Compared machine learning models using scikit-learn and evaluated them with accuracy, precision, recall, and F1-score.

## Tools Used
Python, Pandas, NumPy, Matplotlib, Seaborn, scikit-learn, Jupyter Notebook
